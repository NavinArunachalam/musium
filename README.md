# Virtual Museum Platform

An interactive virtual museum experience built for college students, featuring smooth navigation across digital exhibits.

**Live Demo:** [Virtual Museum](https://virtualmuseum-six.vercel.app/explore)

## Overview

A web-based virtual museum that lets users explore exhibits interactively from any device, designed with a focus on smooth navigation and an engaging, responsive UI.

## Features

- 🖼️ Interactive exhibit exploration
- 🧭 Smooth navigation between sections
- 📱 Fully responsive design (desktop and mobile)
- 🚀 Deployed live on Vercel

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Deployment:** Vercel

## My Role

- Designed and built the interactive exhibit navigation experience
- Built a responsive UI optimized for both desktop and mobile
- Deployed and maintained the live version on Vercel
